## How to play

이동: `A` `D`  
점프: `W`  
앉기: `S`  
슬로우 모드: `Shift`  
구르기(무적): 이동 중 `S`  
공격: `Left mouse`  
아이템 습득/던지기: `Right mouse`  
재시작: `R`  
녹화 영상 재생속도 감소: `A`  
녹화 영상 재생속도 증가: `D`  
녹화 영상 일시정지 토글: `Left mouse`  
사망 시 녹화 영상 빨리감기: `Left mouse`  
클리어 시 녹화 영상 스킵: `Right mouse`
